Common Masters is a function which helps the user to set up their own Master data as required for the system to work correctly.

Common Masters List Screen

In this section, the user can see the already created Common Masters. They can choose to search for a specific master using the master name or its code via the search bar, or can also filter between active and inactive Common Masters.

Edit: The user can also access the ‘Edit’ option by clicking the 3 dot option in the grid of the already created Common Master. Over here, the user can edit the master name, code or status.

Add New Master

By Clicking ‘Add New Master’ on the top right corner of the screen, the user can create a new master for a particular master type. When this happens, the user will select the master type and click on ‘Add New Master', enter a master name and code, choose whether it will be active or inactive, and save the newly created master by clicking ‘Add’.

Users can select the common masters created based on the type while creating the products from the product master.
