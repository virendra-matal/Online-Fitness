EDC Configuration

EDC Configuration is functional and will have provision of configuring the EDC machines through which POS Transactions can be created.  This functionality basically provides provision for swipe/insert payment cards for payment transactions. The EDC Configuration is a provision that can be used by HO users. EDC Configuration provide functionality to configure templates defining general properties. 

To access EDC Configuration: Login into Unify-> Store Management-> EDC Configuration.

EDC Configuration is functional, the system have following options on clicking the menu.

  • Edit: System will allow editing the existing records and values can be edited with new values.
  • Save: System allows to save the new or edited records.
  • View: System will display the existing records on click of search option.
  • Discard - This option will allow you to Discard the Changes done on edit mode.

EDC Configuration

To create a EDC Configuration click on – Edit  option– Users have provision to edit existing  EDC configuration

EDC Configuration List view

 List of saved EDC Configurations will be displayed.

  • EDC Name: This column will display the unique identification Name of EDC machine as defined in EDC Master.
  • Integration: This column will display the values as Yes or No.
  • Merchant Id : This column will display the Merchant Id defined of EDC.

 

Below are the fields where users can define for EDC Configuration:

  • EDC Integration – System have provision to set the EDC machine integration to payment mode for tender transactions. System provide selection of values “Yes” and “No”. On selection of Yes system enable remaining mentioned fields to define values.
  • EDC Swipe Timeout – System have provision to define timeout duration for EDC machine configured on response mode.
  • EDC Card selection– System have provision to select EDC Card Agency manually or automatically. On selection of “No’, card agency and issuing bank will be auto set and on selection of “Yes” cashiers need to mention the details manually.
  • Allow EDC printing - System have provision to print the invoice, to print the invoice “Yes” option to be selected. If the same is not required to print then option “No” can be selected.
  • Default Card Name – System have provision to mention card holder name.
  • Default Card Expiry Date – System have provision to mention expiry date and if the same is not mentioned, system will set default date as 12/99.
  • Merchant ID – System have provision to mention the payment service provider id.
  • Field Name - System have provision to configure the required field for EDC.Name of field to be configured to be mentioned.
  • Field Value – System have provision to mention the value of field name mentioned in Field Name field.
