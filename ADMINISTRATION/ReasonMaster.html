This function allows the user to create reasons that can be assigned for returns, cancellations, rejections and more. This helps understand the reason for the exception taking place. Currently in Ordazzle, the following Reason Categories are available:

Sales Return

Order Cancellation

Order Release

Order Reject

Order Allocation

Shipping Label Cancel

Order Delivery Fail

Order LIT

Stock Out

Stock In

Order Cancellation Accept

Order Cancellation  Reject

Sales Return Intimation

The Reason Master screen allows the user to see the category, the user defined name, and the status of the reason. 

Search By: Users can also search for specific reasons by Reason Category or by Name.

Edit:This option allows the User to edit existing reasons created for a specific category. 

Users can select any of the Reasons from the main screen list and click on the edit option from 3 dots. The user should be able to edit the category, name and status of the reason.

Add New Reason: By Clicking ‘Add New Reason’ on the top right corner of the screen, the user is able to create a new reason for a particular Reason Category. When this happens, the user will select the category, enter a reason name, choose whether it will be active or inactive, and save the newly created reason by clicking ‘Add’.
