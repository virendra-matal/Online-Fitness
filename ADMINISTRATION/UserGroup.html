User Group-WIKI-ETP-Unify

Use Cases and Capabilities of User Group:

User groups in retail software play a crucial role in managing and personalizing the working experience of staff members and employees.

We can create user groups for different employee roles, such as cashiers, store managers, and inventory specialists. Assign specific access permissions and functionality based on these roles.

The User access can be allocated in groups instead of allocation of the access to single users individually, which is time consuming.

The User group Management module in ETP unify provides the service to add groups and allocate set of User Management rights so that its easier to allocate the same.
This module can be used as a template to create user in user management module, when the user group is selected in the User Management module, the modules and access already assigned in the User group will be pre-selected.

User Manual of User Group:

The User can navigate through ETP Unify→Administration→User Group.

User Group List Screen:

When the User opens the User Master, the User can view the card view list of all the already available User groups, if no records available then the screen will be blank.
Each card will have the below,
User Group Name - The Name assigned to the User group on creation.
User type-Will be the User type selected at the time of creation.
3 Dot Button - The 3 dot button will have 2 options, Edit and View, When edit is clicked, then the Create screen will open with the details of the selected card already pre-Populated.
The Search box will allow the user to search based on Group Name.
Drop down for User types, When the User clicks on any one of the User types, the List screen will show the type of user only in the List screen.
The types of Users available are Store, Enterprise,Customer and Vendor.
Status - The User can either select Active or Inactive or All statuses, based on the selection the List screen will show the active and inactive user groups available in the List.
The User can click on the plus button in the List Screen to enter into the create screen.

Create Screen for User Group:

User Group* Here the User can enter the User group name, Eg - Store Managers for whom the group is being created.

User Type* In this Drop down, the User can select the Type of User group, we are going to create, it may be any of the below types - Enterprise, Store,Vendor or Customer.

Active/Inactive ToggleThe User can use this toggle button to activate or deactivate a user management record.

All Access
This is a Check box that can be clicked, when the User being created should have all the available Groups,Companies, Stores and Channels and Functionality. If this check box is checked then the Over ride User group access is not selected and is uneditable.
Only when the All access check box is unchecked then the below options will be
available for selection
 


Add Company
The User can click on this button and the groups available under this login will be shown in the Left panel for selection as a multi selection option, when the User selects a group, the all the companies available under the group is shown below add company option.The User can Multi-select the companies that are needed to be given access for.


Add Channel and Node
When the User clicks on this button the Channels available under the companies previously selected will be shown with the company names on the left side bar. The User can multi select the Channels and Nodes needed under each company from the left side bar.
 


 


Add Functionality
When the User clicks on this button all the functionalities available with ETP Unify will be shown as Menus, Submenus and Functions. The User can multi-select the Menus,submenus and functions needed for the User being created.
 

Now the user can click on the save button to save the User Management record being created.
