The API Dashboard Feature in Ordazzle helps users monitor API performance.

There are two types APIs which would be visible in this screen as mentioned above:

Internal: These are Ordazzle's internal APIs.

External: These are the APIs which the user can create from the API List module. 

API DASHBOARD SCREEN

There is an option to choose Public or Private APIs which the user creates in the 

Request Count: Number of requests to the API 

Success Count :Number of requests which were successfully executed.

Success Rate : Success Rate of the single API requests. 

There are 4 cards on the screen which the user can use to monitor the performance of the APIs over time. 

Today

Last 7 Days

Last 30 Days

Last 90 Days

Searching API Using Filters

The user can select from Internal or External APIs using the radio button. A dropdown list will appear with all services based on the above selection. 

Users can select a particular API to check the performance of the service or API. Users can also select a card from bottom 4 cards to see the relevant chart for the selected service or API.

Reset/ Clear selection

Users can use the refresh or clear button at the top right corner to clear the selection and see the default view of the page.
