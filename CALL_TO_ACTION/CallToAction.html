The Call to Action module informs users when action is required in Ordazzle. It covers Product, Inventory, Orders, Logistics, Channels, and other Services. The user can choose a specific row which requires action, and by clicking on the ‘Action’ button, they will be redirected to the page where the action can be taken. This is all restricted based on the user’s access rights, meaning that if they have only been given access to Product and Inventory for certain nodes, they will only be able to see and process the Product and Inventory points specific to those nodes.

Call to Action List Screen

In this section, users can see module wise tickers and can choose a specific Call to Action, which will redirect them to the page where the action can be taken. Users can apply filters to see All, Active, Snoozed and Dismissed action points.

The module can also be filtered based on priority.

Action: When User wants to take an action on a particular point, the user can click on Action, which will navigate him to the page where the action can be taken.

Snooze/Dismiss:  The user can snooze actions for a day, or for 7 days.

The user also has the option to dismiss an action, in case they do not want to take any action on that point

They can bring any snoozed or dismissed action points back to the Call to Action screen and then act upon it accordingly. They can do so by changing the filter to see snoozed or dismissed actions, and accordingly change their status
