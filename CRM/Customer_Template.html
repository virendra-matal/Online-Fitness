Customer Template

Customer Templates help you to have a set of predefined attributes for different types of customer. 

For example: If users are creating a customer type as Employee or Corporate , then users can select a Template which is created for Employee or Corporate and defined template attributes will be pre-populated like default status of a Customer, Customer group etc.

You can select any existing customer template on the list screen and you will have the following options against it on click of three dots:

  1. View - This option will take you to the customer template creation screen with pre-populated data defined during creation in read only mode.
  2. Edit - This option will open up the customer template for you to edit.  
  3. Delete - This option will allow you to delete the customer template only if customers are not created against it.

 

Customer Template creation:

You can create a customer template by clicking on the “+” add button available in the main screen on the bottom right corner.

On click of the “+” button below the screen will be visible to create a new customer template.

You will be able to pre-define the generic data in the template which will be populated on the customer creation screen.

Below are Fields & Sections where you will be able to define data:

 

Basic details section:

  • Customer Type : You will be able to select the type of customer for which you want to create a template. For example: Individual, Corporate, Family, Walk-In and Employee.
  • Customer Status: You will be able to select the default status for customers using this field.
  • Customer Template Code: You will be able to define a unique template code in this field.
  • Customer Template Name: You will be able to define a unique template name in this field.
  • Title: You can select the title in this field and the same will be auto populated in customer creation screen post selecting template.
  • Gender: You can select the Gender in this field and the same will be auto populated in customer creation screen post selecting template.
  • Age Group: You can select the Age Group in this field and the same will be auto populated in customer creation screen post selecting template.
  • Marital Status: You can select the Marital Status in this field and the same will be auto populated in customer creation screen post selecting template.
  • Country Of Residence: You can select the Country Of Residence in this field and the same will be auto populated in customer creation screen post selecting template.
  • Nationality: You can select the Nationality in this field and the same will be auto populated in customer creation screen post selecting template.
  • Customer Group:You can select the Customer Group in this field and the same will be auto populated in customer creation screen post selecting template.(This data will be come from common master)
  • OTP Verification: You can select the OTP verification type in this field which will be applicable on customer creation. 
    • Yes : If OTP verification is YES then it is mandatory to verify the customer OTP while customer registration.
    • Optional : If OTP verification is Optional then it is not mandatory to verify the customer OTP, instead you can click on “Save without verification” button while customer registration
    • No : If OTP verification is NO then verification popup itself will not come while customer registration.

 

Communication Preferences

In this section you will be able to pre-select the preferred method of communication for the template and the same will be pre-populated in the customer creation screen post selecting customer type & template.

 

Address Details

In this section you will be able to pre-select the Country, State & City against Shipping & Invoicing address for the template and the same will be pre-populated in the customer creation screen post selecting customer type & template.

 

Professional Details

In this section you will be able to pre-select the Profession for the template and the same will be pre-populated in the customer creation screen post selecting customer type & template.

 

Custom Details

Depending on the specific need,  you will be able to add additional user defined attributes. The Attributes could be listed with one or multiselect options, text field, number, checkbox & date. So these additional fields will be available in the customer creation screen in the Custom details section.

 

Fields Configuration

In this section of Customer Template you will be able to set up the fields which are relevant to customer creation of the current customer template and decide which field to keep editable or only show or mandatory or hide. For example, you will like to have a Customer Group for your Customers as ‘Gold member’, you can set this up in customer template and then keep this field as show and not editable.
