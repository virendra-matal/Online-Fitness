This module allows the user to integrate with marketplaces based on region, such as Lazada, Shopee, Zalora, Blibli, Tokopedia, etc., as well as integrations with web-stores such as Magento and Shopify. The user simply needs to provide their login details for the pre-integrated marketplaces, and can then manage all future transactions from these marketplaces on Ordazzle.

The user can get to the Channel Integration screen by going to ‘Connections’ in the menu, and selecting ‘Marketplace Integration’ or ‘e-Commerce Integration’.

Channel Integration List Screen

On this screen, users can view the pre-integrated channels. They can search for channels by their name using the search bar. Users can also see the list of integrated channels country-wise by selecting a country on the left side of the screen.

Channel Creation

Once the user reaches the Channel Integration screen, they will see a list of countries on the left hand side. The user can click on one of these countries where they can either set up a new channel account, or manage an existing one. In order to create a new channel, the user needs to click on the ‘+’ icon.

Over here, a list of pre-integrated channels will appear based on the country selected on the left hand side. Users must drag the channel they want into the ‘drag & drop’ section at the bottom of the screen. From here, the user simply needs to click on the channel and fill in the information accordingly.

Store Name: Users can assign a name for the channel.

Fulfillment Node: Users select fulfillment nodes from which their orders from this channel can be fulfilled. Stock allocation will take place at these nodes when an order is placed on this channel. The user can select a default node that all orders will be routed to, and also decide whether stock should be shared across nodes, or be node specific, by checking or unchecking ‘Send Aggregate Stock Across the Nodes’

For “Shopee”- Users need to input AddressID as the channel warehouse code on the Fulfillment Node selection window.

And to get the Address ID, a user need to go to shopee seller center and in setting user should click on edit address with F12 tab, here the user can see the address ID.

Shop Applicable Product Category: Users can select respective product categories applicable for the respective channel shop. Users will be allowed to select multiple categories for the respective channel shop. 

Shop Applicable Brand: Users can select Brands applicable for the respective channel shop. Users will be allowed to select multiple brands for the respective channel shop.

Based on the selected brands these channel shops will be displayed while listing the Product on Marketplace

MP Seller Types: Here MP seller type can be selected from the options available in the dropdown. MP Seller Type will vary from one channel to another. Users need to select the same ‘Seller Type’ he has selected while creating an account on the Marketplace. Capturing MP Seller Type will help for calculating MP commission.

Auto Allocate Order to Default Node: Users can mark this field as ‘Yes’ or ‘No’. This can be selected as ‘Yes’ to allocate Order to default Node if the Smart routing has not been set up.

Default Jobs & Email Templates Setup: Users can mark this field as ‘Yes’ or ‘No’. Users can select ‘Yes’ if they want a system to Auto-create Default Jobs (background jobs related to syncing order, product and inventory information to marketplace and webstores) as well as Default Email Templates.

Auto Stock Approval: Users can mark this field as ‘Yes’ or ‘No’. The order status will be directly updated as RTP (Ready to Pick) if this option is selected as ‘Yes’. This option will also have an impact in Stock allocation.

Start Date: Users can select the Start Date from the Date picker.

After the user has filled in the above information, they can click on the ‘Connect’ button. For every channel there will be different steps and fields for integration. The user needs to input channel specific fields such as Username, Password, API Key etc to establish the connection between the Channel and Ordazzle.

My Marketplaces 

After creating a channel, the user can see all created channels on the ‘My Marketplace’ screen. Basic information like Shop Applicable Brands, MP Seller Type, Default Jobs & Email Template Auto Stock Approval status and Channel Activation Date will be displayed here. 

The user can also choose to edit, disconnect, Import Product/Orders from MP, Product Mapping, or adjust job schedules of these channels by clicking the three dot option.

Edit: The user can edit any fields for a channel.

Disconnect: The user can disconnect a channel account, this will stop all synchronization with the channel; no new orders will be received and no new products will sync from the channel to Ordazzle.

Job Schedule: The user can see background jobs running for the channel in order for channel integration to work successfully.

Product Import from MP: The user can sync products from the channel to Ordazzle in case there are any products created on the channel before connecting itl to Ordazzle

Order Import from MP: The user can sync Order history for selected time span from the channel to Ordazzle

Product Mapping: The user can map the seller sku with product code if the seller sku and product code is different for the same Product after importing products from the channel to Ordazzle.

Amazon Subscription:

The “Amazon” Subscription process is similar to the above discussed steps, but there are few additional steps that need to be performed.

Amazon Subscription Screen:

 A new field “Seller Partner ID” is on the subscription page

Rest of the fields are as it is which we have in the subscription page.

Step 1: On selecting Amazon Channel User needs to enter all the required fields and click on Connect.

Seller Partner ID user will get from Amazon Seller Center 

(To get Seller Partner ID >> Login to Amazon Seller Center >> Account info >> Your Merchant Token)

Step 2: On connecting we will receive a message “Your Seller Channel has been created successfully, please authorize our "Ordazzle App" from your Seller Central account (Find APP >> Services ) to activate your channel”

Here new status will be assigned for the channel as “Unauthorized” Channel shop and it will be denoted by “Auth. Pending”  for the field “Activated on” at card view.

Step 3: User will login into Amazon Seller center and then from the menu he needs to select “Partner Network” and then inside it we need to select submenu  “Find App and Services” , on this page user will search Ordazzle from search box and he will see  Ordazzle app. On selecting Ordazzle app, user will see an “Authorize” functional button at top right corner of page. 

Step 4: User will click the “Authorize” button and then he will be redirected to the Ordazzle login page, on successful login a user seller channel will be activated in Ordazzle.
