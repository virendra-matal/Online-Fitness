Fulfillment Node Master allows the user to define nodes or fulfillment locations (including warehouses and stores ) which hold inventory. These nodes will also be used for collection and deliveries. 

Fulfillment Node Setup list screen

In this section, the user can see a card view of already created fulfillment nodes. They can choose to search for a specific node using its name or its code via the search bar, or can also filter between active and inactive nodes. Additionally, the user can also edit node details by clicking on the 3 dot option. All fields are editable except the Fulfillment Node Code. The 3 dot option also allows the user to simply view the node, where they will get a summary view of the fulfillment node.

Create New Fulfillment Node

By clicking the plus icon in the bottom right corner, the user can create a new fulfillment node. Clicking this icon will take the user to a new screen where they can enter all required details to create a node.

Basic Details: Over here, the user will capture the Fulfillment Node Code (not editable later, please ensure you input the correct code), Node Name, Node Type and the Node Start Date.

Address Info: Over here the user can capture the Fulfillment Node Address information such as Country, State, City, Address, Postal Code and latitude/longitude details

Other Information: Any additional node related information can be captured over here by the user. This includes the Tax Code, Tax from Date, Timezone, Node Priority, Node Ownership, Node Revenue Share and Default Location can be filled in.
