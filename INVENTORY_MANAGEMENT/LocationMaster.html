Location Master allows the user to create Locations within nodes and fulfillment points, for example, approved, inspection, rejected stock locations

Location master List Screen

In this section, the user can see the already created locations nodewise. They can choose to search for a specific location using the Location name or by Stock Zone via the search bar.

Edit: The user can also access the ‘Edit’ option by clicking the 3 dot option in the grid of the already created Location Master. Over here, the user can edit the Location name.

Add New Location

By Clicking ‘Add New Location’ on the top right corner of the screen, the user can create a new location for a particular Node. When this happens, the user will select the Node and click on ‘Add New Location', select the Stock Zone, Enter the Location Name, select the Location Type and save the newly created Location by clicking ‘Add’.
