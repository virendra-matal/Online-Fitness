Use-case of UOM-conversion:

In ETP Unify “ UOM Conversion” are essential in various industries and applications to ensure accurate and consistent measurement and communication of quantities.
In retail, manufacturing, and logistics, products often come in various packaging sizes and units. UOM conversions help maintain accurate inventory counts and track stock levels in different units, making it easier to manage and reorder products.

Capabilities of UOM-conversion In ETP Unify:

Conversion Factors: UOM conversion typically involves the use of conversion factors. These factors define the relationship between different units of measure for the same quantity. For example, 1 meter is equal to 3.28084 feet, so the conversion factor from meters to feet is 3.28084.

Complex Conversions: Some conversions can be more complex and involve multiple steps or intermediate conversions. For example, converting from pounds per square inch (psi) to pascals (Pa) may require converting both the pressure units and the area units.

Currency Conversion: In financial and e-commerce applications, UOM conversion may also include currency conversion to calculate exchange rates and perform financial transactions.

Conversion Tables and Factors: They may provide access to extensive tables or databases of conversion factors for a wide range of units and categories.

Integration: UOM conversion functionality can be integrated into larger software systems, such as ERP (Enterprise Resource Planning) software, to ensure consistency in measurements across different modules.

Functionality of UOM-Conversion In ETP Unify:

The user can navigate through ETP Unify- Product Management- Uom Conversion.

Once a user goes through the Uom conversion the set of UOMS which has been configured will get displayed.

The Search Bar Tab In the UOm conversions Screen will allow the user to search the Mapped UOM with the UOM names which has been configured.

User Can also search the UOM sets based on the Status Filter:

        a.All Status b. Active C. Inactive

All Status- once the user selects the Filter as All Status All types of UOM set will get displayed on the screen.

Active- Once the user Selects the Filter as Active only Active UOM set List will get displayed on the screen.

In Active- Once the user selects the Filter as InActive at that Case only In-Active UOM set List will get displayed on the screen.

On the UOM set Column user can see the Set of UOMS which has been mapped In system.

The Status Column will be available In the Green color.

Once the user Clicks on 3 dots, the user can see the Edit Functionality which allows the user to edit the existing UOM set which has been mapped.

To create a New UOM Set:

Once user clicks on Add Module the Screen will allow the User to add a New UOM Set:

User need to update the Basic details to add a New UOM Set:

UOM Set Name: User needs to enter the New UOM set name such as UOM SET 14.
Active- User need to make sure the Toggle Is in Active mode.

Add UOM Values:

Base UOM Value- User need to Provide the Base UOM value ( Example such as 1)
Base UOM- User need to Provide the Base UOM from the drop down ( Example Such as Roll)
Alternate UOM Value- User need to Provide the Alternate UOM Value ( Example such as 100)
Alternate UOM- User need to select the Alternate UOM from dropdown ( Example Such as Metres)

Than the Conversion Factor will be:
*1 Roll= 100 Meters *
