Store Promotions Calendar

The Store Promotions Calendar sub-menu in the POS screen which allows User to See The promotion calendar for selected store.

When the user clicks on the "Store Promotions Calendar" sub-menu, they are redirected to the Store Promotion Screen in the Retail Calendar. In this screen, the calendar view is set to monthly view.

In the Store Promotion Screen, users can see a visual representation of the promotions scheduled for the selected store for the entire month. Each day on the calendar may display relevant information about the promotions, such as the promotion name, duration, discounts or offers, and any other details. Users can navigate through different months to see past or future promotions by using the calendar controls.

This feature provides an organized and convenient way for store owners or managers to plan and track promotional activities for specific stores, ensuring that they can effectively manage and monitor their promotional efforts.
