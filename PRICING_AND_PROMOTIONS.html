Rounding Off Rules

The purpose of the rounding-off rules master is to round off the prices or other amounts automatically based on the rounding-off rule configured for the company. The rounding-off rule determines whether the number is rounded upwards, downwards, or to the nearest multiple. The rounding-off table defines how the number is rounded off within different ranges. Rounding off is used to express totals as a specific whole or decimal value. It can be used to even odd fractions for smaller sums or to create so-called psychological prices, such as $9.90. 

Impact: 

 The rounding-off rule will be used while creating pricelists and will also be configured in the retail configuration template (Round Off rule property is available in Configuration type -Store Front->General Settings) which will be applied in billing.

List Screen

The list screen of the rounding-off rules master will display the predefined rounding-off rules and the user can activate any one of the available rounding-off rules which can later be used while creating the pricelists and configuring in the retail configuration template.

Whenever a new company is created, by default rounding-off rules will be available upon opening the rounding-off rule master, and by default ‘No Round Off’ rule will be activated for the company. Users can deactivate the existing rule and activate the required round-off rule.
