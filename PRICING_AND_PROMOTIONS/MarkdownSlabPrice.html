Markdown Slab Price

The markdown slab price allows users to define price slabs that can further create markdown price lists. So whenever the user defines the markdown price list and selects the markdown slab against the price list, then the Slab sales price defined in the markdown slab price will become the new markdown price if any product's sales price is available within the defined range.

List Screen

Over here, users can view all the existing SLABS created for markdown price. 

Users can delete and edit the records and also can add new ones.

The users can sort or use the search bar to search existing slabs by the Slab code or Name on the list screen.  

Each record in the list screen has three dots against it and they will have below following options

  • Edit - This option will open up the creation screen with pre-populated data for users to edit, users can edit names and slab range details in this mode.
  • Delete - This option will allow users to delete the slab record.

Markdown price slab creation

You can create a new slab by clicking on the '+’  add button available on the main screen in the bottom right corner.

On clicking the “+” button, a screen will be visible to create a new markdown price slab.

Below are Fields where you will be able to define data:

  • Price List Slab Code: Users will be able to define the unique code for price list slab.
  • Price List Slab Name: Users will be able to define the name for the price list slab using this field.
  • Status: This will be a toggle button to make the price slab active or inactive.
  • Sales Price Slab Range Details 

This section will allow users to create multiple slab ranges for the price list.

Example: Users may say that all products which range from 10$ to 15$ will have a markdown value of 12$.

  • From Value - This field will be used to define the From value for the slab range.(example: 10 USD )
  • To Value - This field will be used to define the To value for the slab range. (example: 15 USD )
  • SLAB Sales Price - This field will be used to define the slab sales price value for the slab range. (example: 12 USD)
  • Add button - This button allows the user to add the defined range under SLAB.
  • Discard button: This button on the creation and edit screen will allow users to redirect to the list screen without saving the changes.
  • Save button: This button on the creation and edit screen will allow users to redirect to the list screen post saving the changes.
