REPORTING USE CASES AND CAPABILITIES

The Reporting module provides users with a comprehensive set of features to create, customize, and manage various reports. Users can access a range of reports, including favorite, shared, and scheduled reports, making it easy to stay organized and track important data. The tool allows for the creation of customized reports, where users can select specific columns, group data, apply aggregates, sort and filter information, and even apply conditional formatting for enhanced visual representation.

To create a customized report, users can begin by selecting the desired columns to include or remove from the report. This flexibility allows for tailored reports that focus on the most relevant data. Additionally, users have the option to group the data based on specific criteria, enabling them to analyze information in a more meaningful way.

Sorting and filtering capabilities are essential for organizing and narrowing down large sets of data. Users can sort data based on specific columns in ascending or descending order. Furthermore, the filtering functionality allows users to set criteria to display only the data that meets certain conditions.

Users can easily filter data based on specific date ranges. The Reporting module also offers conditional formatting options, allowing users to highlight data, making it easier to identify trends or anomalies at a glance.

Once the report criteria are set, users can run the report and view the results. The tool also offers options to customize the header and footer of the report, providing a professional touch and allowing users to include additional information. Users can save their customized reports for future use and access them conveniently. Additionally, the module offers a report scheduler, enabling users to automatically schedule reports for specified intervals.

Column Selection (Add/Remove):

  1. The user has the option to customize their report by selecting specific columns from a pre- defined template.

Report Schedular

The report schedular helps the user to schedule a particular report based on required time intervals and required format (i.e excel/pdf)

  1. While saving the report the user needs to select the dates or last no. days from the date option available in the report customization screen and save the report accordingly.
  2. In the all reports screen the user needs to click on the vertical ellipse that is available for the desired report he wants to schedule.
  3. Click the schedule option and Report schedular popup will appear.
  4. Here the user needs add the emails of the user to which the report needs to be sent (scheduled) through email.(separated by comma)
  5. Select the From date and To date option so as to schedule the report for the desired duration. 6.Select the frequency for which the user wants to see the report ex.Daily,Weekly,Monthly
    • Daily-If user selects daily he just needs to enter the time on which the report should run and the report will be delivered on daily basis.
    • Weekly- If the user selects weekly, here the user needs to select the weekday and time on which the report should run.
    • Monthly-If the user selects monthly,then he needs to select the date and time on which the report should run on monthly basis.
  6. Select the report format Ex. Excel or PDF and click schedule
  7. This scheduled report will be delivered on the entered email addresses as per set frequency.
  8. Run history for such reports can be seen in the Run history screen which can be accessed through schedule report screen.

Other Features

Column Level Sort

The user can sort the data at UI level using column level sorting feature.

  • The user can access the feature by clicking on the column name to rearrange the data in ascending or descending order.
  • This sort will be applied on the pre loaded data only and if the user wishes to apply sort on all the data then the user needs to load all available data by scrolling down.
  • Or the user can download all the data using the download option given at the UI screen and download the file for further analysis.

Column Level Filter

The user can filter the data at UI level using column level filter feature.

  • The user needs to click the filter icon present next to the column name.
  • Select the filter condition (Ex. Contains,Equals etc)
  • Add value based on which the data needs to be filtered. And/or condition can also be used to apply second filter on that column.
  • Now the data will load as per the applied filter.

Date Filters (UI)

  • The user can user the date filters at UI level that are available to run and load all required data based on desired timeline.
  • The user can apply date filters like YTD,QTD,MTD,WTD,Yesterday,Today or can also use custom dates to load the data.

Download

  • The user can download all the available column data using download functionality.
  • User need to click on the download icon available at the top right corner of the UI.
  • The file gets downloaded with desired data and report name


 

Reporting Module Guidelines

  • Report with run report criteria cannot be scheduled.
  • If the user wants to create an aggregated report then he must apply the group by and Aggregate to the applicable columns.
  • Sort, Filter and conditional formatting cannot be applied in UI on aggregated and calculated columns/fields(as they are virtual columns)
  • Run history can only be seen for the scheduled reports.
  • While scheduling the reports in pdf format the user needs to select the orientation based on the number of columns that he has selected.(the report might look distorted if excess

columns are selected)

  • The user needs to select the Dates from the dates option available in the report

customization screen before scheduling the report (to enable the schedule report option)

  • The user needs to load all the data by scrolling down if he wishes to apply/use column level sort and filter functionality.
  • Customize report date filters are for scheduled report only, after opening any report in UI the user needs to apply date filters available in the left pane to load data for desired timeline.
