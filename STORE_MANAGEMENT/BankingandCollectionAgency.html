Banking and Collection Agency

The Banking and Collection Agency list screen allows users to create and manage Bank Accounts and Collection Agencies at the company level. By utilizing this function, users can create bank accounts that enable them to perform various financial activities such as depositing, withdrawing, and making expenses, while also allowing third-party collection agencies to deposit different forms of payments on behalf of the store.

Dropdown filters

Entities: The list screen can be filtered to display only bank accounts and collection agencies, allowing users to manage and track financial transactions related to these entities easily.

Status: With the option to filter by statuses such as active, inactive, or all statuses, the list screen can be further customized to display only the desired information, and when combined with the bank accounts and collection agencies filters, users can efficiently manage and track financial transactions for specific entities and statuses.

Search Filter by Entity Name or Entity ID: Users can easily search the list screen by either entity name or entity ID, allowing for quick and efficient retrieval of specific financial entities.

Three Dots Button: 

The three dots function provides users with a range of functionality, including the ability to edit or view entities and activate or deactivate them as needed. 

View: When clicking the view option for a specific entity on the list screen, users can view all details previously filled for that particular bank account or collection agency, with all fields displayed in a disabled status.

Edit: By clicking the edit option on the three dots, users can modify details for entities that were previously set up at the company level, allowing for easy updates.

Activate/Deactivate: Clicking the activate option allows users to activate previously deactivated bank accounts or collection agencies at the company level while clicking the deactivate option enables the deactivation of previously activated bank accounts or collection agencies at the company level.

New Bank Account Record:

To create a bank account for the purpose of maintaining records of deposit, withdrawal, and expense transactions, simply click on the plus button. You can then click on the bank account label to be redirected to the creation screen where you can fill in the necessary details.

Details on Bank Account Creation Screen:

Account Type: To set up an account type at the company level, users can simply click on the account type dropdown and select the desired option.

Bank Name: To create a bank account, users can select a bank from the bank name dropdown menu.

Account Number: To set up a bank account, users can input the account number for the corresponding account.

Bank Branch: To set up a bank account, users will be able to input the bank branch where the account is located.

Bank ID: When setting up a bank account, users can enter the bank ID, which is the IFSC code unique to a particular bank.

Account Holder Name: In this field, users can enter the account holder's name when setting up a bank account.

IBAN No.: In this field, users can enter the IBAN number when setting up a bank account. IBAN stands for International Bank Account Number, a unique identifier assigned to each bank account and used for international transactions.

Foreign Currency Allowed: Users can select this checkbox to enable or disable foreign currency transactions when setting up a bank account.

Swift Code/BIC Code: In this field, users can enter the SWIFT code or BIC code when setting up a bank account. SWIFT (Society for Worldwide Interbank Financial Telecommunication) code is a unique identifier assigned to financial institutions for international wire transfers, while BIC (Bank Identifier Code) is a similar code used in Europe to identify banks in international transactions.

Country: Users can select the country to which the bank account being created belongs by selecting it from the dropdown menu.

State: To set up a bank account, users can select the state to which it belongs by choosing an option from the dropdown menu.

City: Users can select the city to which the bank account being created belongs by choosing an option from the dropdown menu when setting it up.

Postal Code: When setting up a bank account, users can enter the postal code in this field based on the location to which the bank belongs.

Remarks: Users can enter remarks or additional notes related to the bank account being set up in this field.

Account Applicable For: Users can multi-select the applicable accounts for deposit, withdrawal, and expenses when setting up a bank account, allowing them to use each account for its respective purpose.

Details on Collection Agency Creation Screen:

Collection Agency: When setting up a bank account, users can enter the name of the collection agency in this field, which will be applicable for collecting cash or cheques on behalf of the store.

Collection Agency ID: In this field, users can enter the unique ID for the collection agency when setting up a bank account.

Contact Person: When setting up a bank account, users can enter the name of the contact person in this field.

Country Code and Phone Number: Users can select the country code and enter their phone number in this field when setting up a collection agency.

Email Address: When setting up a bank account, users can enter the email address of the contact person who will be responsible for collecting cash or cheques from the store and depositing them in the bank on behalf of the store.

Identification Proof Number: Users can enter a unique identification number, such as a driving license number, PAN number, Aadhaar number, or other similar identification numbers. 

Collection Schedule: Users can select the collection schedule for cash or cheque collection in this field.

Forms of Collection: Users can multi-select the forms of collection, such as cash, cheques, and demand drafts.

Country: Users can select the country to which the collection agency being created belongs by selecting it from the dropdown menu.

State: To set up a collection agency, users can select the state to which it belongs by choosing an option from the dropdown menu.

City: Users can select the city to which the collection agency being created belongs by choosing an option from the dropdown menu when setting it up.

Postal Code: When setting up a collection agency, users can enter the postal code in this field based on the location to which the bank belongs.

Remarks: Users can enter remarks or additional notes related to the collection agency being set up in this field.
