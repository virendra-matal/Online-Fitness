Customer Circle

This functionality is to allow the admin level user at HO to create Customer Circle criteria for different sets of stores. Customers are created at every store of the company. This generally leads to the creation of multiple customer records for the same individual. To avoid this duplicate records creation, a Customer Circle is required. At the time of registering a new customer with its details it searches for the existing customer database & their Email ID and/or phone number based on the defined Customer Circle criteria for the store.

Customer Circle helps you to define deduplication criteria for customer creation. Defining Customer Circle also helps you to avoid duplicate customer creation.(Unify > CRM > Customer Circle)

While creating the Customer Circle criteria, the user will select the criteria defining properties, also will select the set of stores based on provided store attribute fields.

List Screen

The User will be able to view the main screen of the functionality which will display the list of Circle criteria/rules already created in the system in card view format. This includes all the active (advance created & running) and inactive records. Every card will contain Customer Circle Name, Criteria, Valid from & Status  field values from the saved rule records. Each card will also have an option button & status indication at its top & bottom corners.

Search Option: The user will be able to search by Customer Circle Name and Valid From (YYYY/MM/DD) from the Search field in the List screen.

Status: The User will be able to filter the Customer Circle by changing the status from the status indication as All Statuses, Active, Inactive.

The user can select any existing customer circle and will have the following options against it:

  1. View - This option will take you to the customer template creation screen with pre-populated data defined during creation in read only mode.
  2. Edit - This option will open up the customer template for you to edit
  3. Deactivate - This option will allow you to deactivate the customer circle only if there are no customers created for that customer circle.  
  4. Delete - This option will allow you to delete the customer template only if Customer Circle is deactivated.

Rule Creation Screen

Customer Circle Name: The user will be able to input the Unique Name of the Customer CIrcle in this field. This will accept only 30 alphanumeric characters. (Example : Adidas, Nike etc)

Deduplication Criteria: In this field the user will define that the duplicate customer will be searched on which of the following conditions/criteria of customer details.

Mobile no. Only : This means customer deduplication will be performed only on mobile no.

Email Only : This means customer deduplication will be performed only on Email id.

 Mobile no. and Email : This means customer deduplication will be performed on Email id and mobile no.

Mobile no. or Email : This means customer deduplication will be performed only on Email id or mobile no.

Company: The users will be able to select multiple companies for a Customer Circle using this field. This field will only show the List of those companies for which the user is having access.

This field will also have below two options at bottom of dropdown field:

  1. Select All : On clicking this option all companies will get selected in the dropdown list.
  2. Clear All : On clicking this option all selected fields will get unselected.

Valid From: The User will be able to define the date from which the rule will be applied to the specified stores. Although the status of the rule has to be Active in order to apply it on valid from date. (For example: If valid from date is selected as future date X then customer deduplication will not be done on customer creation today.)

Status: The user will be able to define if the created Circle rule is to apply to the stores as Active or Inactive.

Customer Circle Level : 

This section allows you to select the different levels for creating a customer circle. For example: you can create customer circle at company level

  • At Company Level : This checkbox allows you to create a customer circle at company level.
  • Channel : This field allows you to select single or multiple channels for the customer circle.
  • Store Brand : This field allows you to select single or multiple store brands for the customer circle.
  • Store Concept : This field allows you to select single or multiple store concepts for the customer circle.
  • Store Group : This field allows you to select single or multiple store groups for the customer circle.
  • Store Category : This field allows you to select single or multiple store categories for the customer circle.
  • Ownership : This field allows you to select single or multiple types of ownerships for the customer circle.
  • Mall : This field allows you to select single or multiple malls for the customer circle.
Validate button : 

On click of this button you will be able to view the overlapping stores with other customer circles. 

For example: If you are trying to create a customer circle for Brand: Nike and someone else has created the customer circle at company level then on click of the validate button system will display the overlapping stores having other customer circles. Once you verify the data you can click on the “SAVE” button to save the customer circle record.

Note: 

  1. If you want to create a customer circle for a combination of multiple store attributes. (BRAND, Concept, Group, Category, Ownership & Mall) 

For Example: If you want to create a customer circle for Store brand (Nike) and Ownership (Owned) then on click of the validate system will create a customer circle for stores only for the combination of Brand: Nike & Ownership: Owned.

  1. Post displaying records on grid  section if you do not wish to add the store in the customer circle which you are creating you can simply uncheck the checkbox and click on save, on click of save system will not update the customer circle for unchecked record.
  2. If you are trying to create a customer circle at company level then you will not be able to uncheck the overlapping records on the grid.
  3. If you are selecting multiple companies in the basic details section then you will not be able to select the channel or store attributes for the customer circle.
  4. If you select the checkbox at company level then you will not be able to select the channel or store attributes for the customer circle.
Reset button

This button allows you to reset the customer circle level section. On click of this button all data will be cleared from the level selection field.

Grid section post clicking on validate button.
  • On New customer circle creation this section will populate the overlapping data.
  • On View this section will display the existing records of stores/channels in the customer circle.
