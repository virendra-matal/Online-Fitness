EDC Master

The EDC master in Unify gives users a provision to create EDC in the Stores. The functionality is basically used for payment through swipe/card.

EDCs of Stores are required for POS transactions made using a payment card. The EDC is required in EDC configuration at Store level & counter level, it is also required in Accelerator to define Bank/EDC based promotion. 

Example - Two EDC machines will be assigned to a Single or Multiple stores , which can be configured in the following method.

When EDC Integration is mentioned as “YES” , the creation of the tender system will provide provision for EDC selection , on which details of the card will be retrieved and accordingly the payment method will be reflected through EDC Machine configured.

To access EDC master: Login into ETP Unify -> Store Management -> EDC master.

EDC Master is functional, the system will have options on clicking menu option.

  • Add: System will allow to add a new EDC store , following values can be defined on addition of new record , EDC Name(Unique Identification Name),Commission%( percentage for charges applicable provided by merchant),Tolerance%(percentage for charges applicable on payment mode) and Status ( Current status of record as Active and Inactive) and are mandatory.
  • View: System will display the existing records on click of search option.
  • Edit – System will allow to edit the existing records and values can be edited with new values..  
  • Delete - This option will allow you to delete the EDC added.
  • Discard: System allow to discard the record if the details filled are not apropriate before saving the record on add or edit functionality.
  • Active/Inactive toggle - Users can maintain the Active and Inactive status of the EDC..
  • Save : System allow to save the new or edited records.

EDC Master Creation

To create a EDC click on - Add button. Users have provision to create EDC for a store.

Below are the fields where users can define for EDC.:

  • EDC Name – Unique Identification Name can be defined , which will be reference for Billing..
  • Commission %- System allow to define commission percentage of Merchant Service provider for transactions carried from EDC Machine.
  • Tolerance% - System allow to define tolerance percentage for card network, card type and bank issuing card.
  • Status: System allow to Active or Deactivate the EDC record.

EDC List view

List of saved EDC records will be displayed.

  • EDC Name: This column will display the unique identification Name to which the EDC is assigned.
  • Commission% - This column will display the Commission% to which the EDC is assigned.
  • Tolerance% - This column will display the Tolerance% to which the EDC is assigned..
  • Status - This column will display the active/inactive status of the created target.
