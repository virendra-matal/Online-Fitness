Employee Master

The term "employee master" typically refers to a central database or system that stores and manages information about employees within an organization of Stores and Head offices. It is a comprehensive repository of employee data, including personal details, employment history, job titles, contact information, and other relevant information.

The employee master database is used by human resources departments and other relevant stakeholders to maintain accurate records of all employees within the organization. It allows HR professionals to track and manage employee information efficiently, streamline administrative tasks, and generate reports or analytics related to the workforce.

Store ID - The ETP Unify user can filter out past records by entering Store ID in the search tab. After entering a particular Store ID, all the records will be displayed  for the particular Store as per below - 

Employee ID - The ETP Unify user can search a particular record by entering the employee Id in the search field. After entering the employee Id the display screen will be displayed as below - 

Employee Name - The ETP Unify user can search a particular record by entering the employee name in the search field. After entering the employee name the display screen will be displayed as below -

Employee Type - The ETP Unify user can filter out records by selecting Store or HO as below - 

Status - The ETP Unify user can filter out records by Active or inactive Status as shown below - 

3 Dots - The user can Edit, View or delete an existing record by clicking on the 3 dots against the record. 

Create New Employee Record - The ETP Unify user can create a new single employee record by clicking on the Plus Icon and clicking Create -

 

After Clicking on Create the user will be directed to a new screen and enter the details - 

Upload Profile Picture -  The ETP Unify user has to click on the Upload profile picture and upload the file ( Picture of Employee )  from the system. After successfully uploading the file, the user will be able to view the picture of the employee.

Active / Non-Active - The user can choose to make the Employee record as active or inactive by clicking on the active/Inactive toggle button.  

Employee Type - The user can select HO or Store from the employee type while creating a new employee record.

Employee Store - If the user has selected Store as employee type, then all the active Nodes ( Stores ) under the company will e shown in the Employee Store.

And if the user has selected HO as employee type then the Employee Store will be disabled. 

Employee Id - The user needs to enter a unique employee id for creating a new employee record which should not be a duplicate value in the database. 

Employee Hierarchy ID - The user has to select employee hierarchy when creating an employee record. The details will be fetched from the hierarchy module.

First Name - After selecting the prefix, the user needs to enter the employees's First name.

Middle Name -The user needs to enter the employee's middle name.

Last Name - The user needs to enter the employee's last name.

Gender - The user has to select the employee's gender from options like Male, Female, and Other.

Date of Birth - The user has to select the employee's Date of birth from the calendar.

Wedding Anniversary - The user has to select the employee's Wedding anniversary from the calendar. 

Is Sales Person - The user has to click on the check box if the employee is a Salesperson.

Mobile Number - The ETP Unify user after selecting the ISD Code, needs to enter the mobile number ( numeric value ) of the Employee. 

Email ID - The ETP Unify user has to enter the email ID of the employee in the field.

Address Line 1 - The user can enter the Address of the employee in the Address Line 1 field.

Address Line 2 - The user can enter the Address of the employee in the Address Line 2 field.

 

Biometric ID - A unique ID that will be used by the employee for biometrics needs to be entered in the field.

Employee on Role - The user has to click on the check box if the employee is a On Role employee..

Assign Target - The user has to click on the check box if the employee has to be assigned Target..

Date of Joining - The user has to select the employee's Date of Joining from the calendar. 

Date of Leaving - The user has to select the employee's Date of Leaving from the calendar.  

Department - The user can select the department type the employee is associated with. The options coming for the department will be coming from the common masters. 

Employee Group - The user can select the employee group the employee is associated with. The options coming for the department will be coming from the common masters.  

Qualification - The user has to enter the Qualification details of the employee in the qualification field.

Training Level - The user has to enter the Training level details of the employee in the Training level field.

Conduct -  The user has to enter the Conduct details of the employee in the Conduct field.

Exit Interview - The user has to enter the exit interview details of the employee in the exit interview field. 

After entering the details of the Employee, the user needs to click on the Save tab in the right corner of the screen. After clicking on Save the user will be directed to the Employee master list screen and the record will be displayed on the list screen.

Download / Upload - The ETP Unify user can do a bulk creation of new employee records by using the download and upload process. The user needs to click on the plus icon and click download. After clicking on download a CSV file will be downloaded to the system. The user needs to fill in the employee details in the file and then click on upload and browse the saved file and click on the file to be uploaded. Once the file is uploaded the message will be shown that the details have been successfully updated. The user then will be able to see all the records on the Employee master list screen.
