Number Series Template

Number Series can be defined as a sequence formed out of numbers which is used as unique identification code to identify type and transactions such as general ledger accounts, customer and vendor accounts, invoices, external documents and so on.

Number Series template is used to configure templates which are further used in the store config group instead of creating individual number series for every store.

Example: Users  can set up one code for numbering customers, another code for numbering sales invoices, and another code for numbering documents in general journals. 

After users have set up a code, users must set up at least one number series line. 

The number series line contains information such as the first and last number in the series and the starting date. The series will be used consecutively, starting each series on the respective starting date.

List Screen

Over here, users can view all the existing Number series templates created. 

Users can delete and edit the records, and also can add new ones.

Users can sort or use the search bar to search existing records by the number series template code or Name on the list screen. 

Each record in list screen has three dots ‘

’,  against it and they will have below following options

  • View - This option will open up the creation screen with pre-populated data for users to view, in read only mode.
  • Edit - This option will open up the creation screen with pre-populated data for users to edit, users can details in this mode.
  • Delete - This option will allow users to delete the record only if it is not used in billing transactions.

Number Series Template creation

Users can define a new charge by clicking on the ‘

’  add button available in the main screen on the bottom right corner.

On click of the “+” button , a screen will be visible to create a new number series template.

Below are Fields where users will be able to define data:

  • Number Series Template Code : The unique code for number series template. 
  • Number Series Template Name : The name of a number series template. 
  • Usability Mode : It identifies the mode against which a number series template is created. There are two modes (Production / Training )
  • Number series creation for transaction :

Below are the fields required for number series configuration for different type of transactions:

  • Prefix 1 : This indicates the part of the series to be present at starting place of Number Series
  • Prefix 2 : This indicates the part of the series to be present at second place of Number Series. This field has predefined options as Store, Year & NA
  • Prefix 3 : This indicates the part of the series to be present at third place of Number Series. This field has predefined options as Store, Year & NA
  • Start No : This indicates the starting number of the variable part of the number series.
  • End No : This indicates the last number of the variable part of the number series.
  • Last Number Used : This indicates the last performed transaction number for the series. So the series will use the next number of upcoming transactions.
  • Display : This represents the sample number (largest possible) based on selected/entered parameters for the number series type. 

(i.e. If user has defined below - 

  • CN (Prefix 1)
  • STORE (Prefix 2)
  • 2022 (Prefix 3)
  • 1 (Start no) 
  • 99999999 (End no)
  • 98 (Last number used) 
  • Then the value will be displayed as CNMBO202200000098 in display columns.
