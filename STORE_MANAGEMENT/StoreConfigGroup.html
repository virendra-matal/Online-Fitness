Store Configuration Group

The primary objective of this master is to enable the configuration of individual stores or groups of stores using predefined master templates. Business users have the ability to create specific configuration groups for various store attributes, allowing them to associate configuration templates with each group. These attached templates will solely apply to the stores associated with the corresponding store attribute for which the store configuration group was created. To access this functionality, users can find the master under the ' Store Management ’ menu.

  • The following templates can be attached to a store configuration group
    • Number Series - Production
    • Number Series - Training
    • Retail Configuration - Store Operations
    • Retail Configuration - Store Front
    • Retail Configuration - Counter
    • Retail Configuration - Invoice
    • Retail Configuration - Credit Note
    • LookUp
    • Retail Configuration - Orders
    • Retail Configuration - Order Invoicing
    • Retail Configuration - Sales Return

LIST SCREEN:

  • On accessing the Store Configuration Group from the ‘Store Management’ menu, a list screen will be shown with the Store Attribute selection, Mapped, and Unmapped sections.
  • Unmapped Section: Upon selecting a store or store attribute, the stores or attribute values associated with that specific store attribute will be showcased in the unmapped section. Here, users have the option to click on the 'ADD TEMPLATES' button, enabling them to create a store configuration group for each individual store or store attribute value.
  • Mapped Section: The store configuration groups created for the store attributes or store will populate under the mapped section which determines that the templates are configured for the store/store attribute.
  • Store Attribute Name: Under the mapped section, the name of the attribute type selected from the dropdown will be displayed.
  • Stores: Number of stores belonging to the store attribute value, The count will be 1 in case attribute type is selected as Store since individual stores will be shown.
  • Name: Name of the Store configuration group created will be displayed here.
  • Users can also search for the existing store configuration groups or attribute values from the search box available at the top right corner.
  • For each store configuration group created, there is a 3 dots button which has the option to EDIT. Given the user has access to this option - they can edit an existing store configuration group and modify the attached templates as required.

Store Configuration Group Creation Page:

  • Users can create a store configuration group using the ADD TEMPLATE button available against each store or store configuration group, which will open up the creation page where users can select the templates for each configuration type.
  • Name: Users can enter the name of the store configuration group to be created. This is a mandatory alphanumeric field where users can enter up to 50 characters.
  • Store Attribute Name: The name of the attribute type of the attribute selected. This is a read-only field.
  • Store Attribute Value: The name of the attribute value of the attribute type selected. This is a read-only field.
  • Under the store configuration template assignment section, users can select the required templates to be attached for the following configuration types:
    • Number Series - Production
    • Number Series - Training
    • POS Configuration - Store Operations
    • POS Configuration - Store Front
    • POS Configuration - Counter
    • POS Configuration - Invoice
    • POS Configuration - Credit Note
    • Lookup
